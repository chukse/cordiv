This directory contains the ddG data generated from FoldX and associated scripts.

Contents:
	foldx_setup/
		Contains the repaired_pdbs used in FoldX

	FoldX_results/
		Contains the files generated by the Position Scan method in FoldX
	
	foldx_ddG/
		Contains a file for each protein with the ddG data for each residue		
	
	scripts/
		Contains the scripts to generate runfiles, pdb lists, etc to run the FoldX application
		It also contains a script that combines the raw FoldX output into a single file for each protein
